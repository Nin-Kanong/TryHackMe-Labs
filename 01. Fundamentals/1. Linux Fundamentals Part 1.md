## Task 2: A Bit of Background on Linux

### Where Linux is used

Linux is used in many everyday technologies, such as:
- Websites and servers
- Car entertainment systems
- Shop checkout machines (POS systems)
- Traffic lights and industrial machines

It is popular because it is lightweight, powerful, and open-source.

### Flavours (Distributions) of Linux

- **Linux** is not just one version. There are many different versions called distributions (distros), built for different purposes.

Examples:
- Ubuntu
- Debian

Some are made for servers, others for personal computers.

## Task 4: Running Your First few Commands

### 1. Terminal
The ``terminal`` is a text-based tool where you type commands to control Linux.

Example Prompt:
````
tryhackme@linux1:~$
````


### **echo** command
`echo` print text on the screen.

Example:
````
echo Hello
````
