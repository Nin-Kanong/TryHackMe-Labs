## Task 1: Windows Editions

### Windows started long ago
- Windows has existed since 1985
- It became the most used operating system for:
   - Home computers
   - Companies
   - Businesses

Because many people use Windows, hackers often target it with:
- Malware
- Viruses
- Attacks


### Windows XP became very popular
Windows XP was:
- Stable
- Fast
- Used for many years


### Microsoft released Windows Vista
After XP came Windows Vista.

Problem:
- Many bugs
- Performance issues
- Users disliked it

Result:
- Vista quickly failed.

Think:
````
Windows XP → 👍
Windows Vista → 👎
````




### Microsoft ended support for XP
Microsoft announced:
````
No more updates for XP
````
Companies became worried because:
- Hospitals used XP
- Businesses used XP
- Hardware depended on XP

They needed a replacement.




### Windows 7 arrived
Windows 7:
- Fixed Vista problems
- More stable
- Faster
- Users loved it

Think:
````
XP → Vista ❌ → Windows 7 ✅
````



### Windows 8 came
Windows 8:
- New interface
- Short lifespan
- Many users disliked the changes

Like Vista, it wasn't very successful.

### Windows 10 arrived
Windows 10 became:
- More stable
- Widely used
- Long-term supported

For years it was the main Windows version.


### Windows 11 became current

Now:
- Windows 11 is Microsoft's current desktop OS.

Main editions:
- **Home** → personal users
- **Pro** → business users

### The Taskbar
Some of the components are enabled and visible by default. The Toolbar (6), for example, was enabled for demonstration purposes.  

If you're like me and want to disable some of these components, you can right-click on Taskbar to bring up a context menu that will allow you to make changes.
<img width="358" height="476" alt="image" src="https://github.com/user-attachments/assets/66a3c8ae-e043-4238-a82b-600e677dcd3e" />

Any apps/programs, folders, files, etc., that you open/start will appear in the taskbar. 

<img width="631" height="282" alt="image" src="https://github.com/user-attachments/assets/ccab70d1-a8d2-4a96-89d8-992a1352e9fa" />

Hovering over the icon will provide a preview thumbnail, along with a tooltip. This  tooltip is handy if you have many apps/programs open, such as Google Chrome, and you wish to find which instance of Google Chrome is the one you need to bring in to focus. 

When you close any of these items, they will disappear from the taskbar (unless you explicitly pinned it to the taskbar). 

The Notification Area

The Notification Area, which is typically located at the bottom right of the Windows screen, is where the date and time are displayed. Other icons possibly visible in this area is the volume icon, network/wireless icon, to name a few. Icons can be either added or removed from the Notification Area in Taskbar settings.
