## Task 2: System Configuration and Advanced System Settings

#### What is System Configuration (MSConfig)?

**MSConfig** = **System Configuration Utility**

System Configuration is a Windows troubleshooting tool used to control:
* How Windows starts
* Which services run
* Boot options
* Troubleshooting tools

It is mainly used by:
* System administrators
* IT support
* Cybersecurity students
* Troubleshooting Windows startup problems


### How to Open MSConfig

#### 1. Start Menu
Press **Windows key** -> Type **msconfig** -> Click **System Configuration**


#### 2. Run Dialog
- Press **Win + R** -> Type:
```
msconfig
```
- Press Enter

We need **Administrator rights**.




---

### General Tab

This tab controls **what Windows loads during startup**.

#### Options

| Option             | Meaning                                      |
| ------------------ | -------------------------------------------- |
| Normal Startup     | Load everything normally                     |
| Diagnostic Startup | Load only basic Windows services and drivers |
| Selective Startup  | Choose manually what loads                   |

---

#### Think of it Like This

| Mode       | Example                       |
| ---------- | ----------------------------- |
| Normal     | Full computer starts normally |
| Diagnostic | Safe troubleshooting mode     |
| Selective  | Custom startup                |

---

### Why Use It?

Used when:
* Windows crashes
* Malware affects startup
* Drivers cause problems
* Slow boot issues

---

### Boot Tab

Controls how Windows boots.

---

#### Important Options

| Option      | Meaning                           |
| ----------- | --------------------------------- |
| Safe boot   | Starts Windows in Safe Mode       |
| No GUI boot | Removes Windows loading animation |
| Boot log    | Saves startup logs                |
| Timeout     | How long boot menu waits          |

---

#### Safe Boot Types

| Type            | Purpose                       |
| --------------- | ----------------------------- |
| Minimal         | Basic Safe Mode               |
| Network         | Safe Mode with internet       |
| Alternate shell | Safe Mode with Command Prompt |

---

#### Example

If Windows is broken:
Open ``msconfig`` -> Go to ``Boot`` -> Check ``Safe boot`` -> ``Restart PC``

Now Windows starts in Safe Mode.

---

### Services Tab

This tab shows all Windows services.

A **service** is a background program.

Examples:
* Windows Update
* Antivirus
* Print Spooler
* DHCP Client

---

#### What You Can Do

| Action          | Purpose                   |
| --------------- | ------------------------- |
| Enable service  | Start service on boot     |
| Disable service | Stop service from loading |

---

### Important Checkbox

#### “Hide all Microsoft services”

Very important for troubleshooting.

Why?

* Prevents disabling critical Windows services accidentally.

---

### Example Troubleshooting

If a third-party app causes crashes:
- Hide Microsoft services
- Disable remaining services
- Restart
- Test system

This is called a **Clean Boot**.

# 4. Startup Tab

This tab normally manages startup applications.

BUT:

On newer Windows:

* It redirects to Task Manager.

On Windows Server:

* It may appear empty.

---

# Why Empty on Windows Server?

Windows Server handles startup differently.

You usually check startup apps using:

```cmd
shell:startup
```

---

# How to Open Startup Folder

1. Press **Win + R**
2. Type:

```cmd
shell:startup
```

3. Press Enter

This folder contains:

* Shortcuts
* Programs
* Scripts
* Apps that run after login

---

## Common Red Team / Blue Team Importance

Attackers often place malware here for persistence.

Security analysts check:

* Startup folder
* Registry Run keys
* Scheduled tasks
* Services

---

# 5. Tools Tab

This is like a shortcut menu for Windows admin tools.

---

## Examples of Tools

| Tool               | Purpose                |
| ------------------ | ---------------------- |
| Event Viewer       | View logs              |
| Command Prompt     | Run commands           |
| Registry Editor    | Edit registry          |
| Task Manager       | Manage processes       |
| System Information | Hardware/software info |

---

## Selected Command Section

When you click a tool:

* MSConfig shows the command used to launch it.

Example:

```cmd
eventvwr.msc
```

opens Event Viewer.

---

# Why This Matters in Cybersecurity

MSConfig helps:

* Malware analysis
* Persistence detection
* Service investigation
* Troubleshooting compromised systems
* Safe Mode recovery

# Real Practical Example

## Scenario:

Computer becomes very slow after installing software.

## Troubleshooting:

### Step 1

Open:

```cmd
msconfig
```

### Step 2

Go to:

* Services
* Hide Microsoft services
* Disable suspicious services

### Step 3

Go to:

* Startup
* Disable suspicious startup apps

### Step 4

Restart PC

If issue disappears:

* One service/startup app caused problem.
