## Task 2: System Configuration and Advanced System Settings

#### What is System Configuration (MSConfig)?

**MSConfig** = **System Configuration Utility**

System Configuration is a Windows troubleshooting tool used to control:
* How Windows starts
* Which services run
* Boot options
* Troubleshooting tools

It is mainly used by:
* System administrators
* IT support
* Cybersecurity students
* Troubleshooting Windows startup problems


### How to Open MSConfig

#### 1. Start Menu
Press **Windows key** -> Type **msconfig** -> Click **System Configuration**


#### 2. Run Dialog
- Press **Win + R** -> Type:
```
msconfig
```
- Press Enter

We need **Administrator rights**.




---

### General Tab

This tab controls **what Windows loads during startup**.

#### Options

| Option             | Meaning                                      |
| ------------------ | -------------------------------------------- |
| Normal Startup     | Load everything normally                     |
| Diagnostic Startup | Load only basic Windows services and drivers |
| Selective Startup  | Choose manually what loads                   |

---

#### Think of it Like This

| Mode       | Example                       |
| ---------- | ----------------------------- |
| Normal     | Full computer starts normally |
| Diagnostic | Safe troubleshooting mode     |
| Selective  | Custom startup                |

---

### Why Use It?

Used when:
* Windows crashes
* Malware affects startup
* Drivers cause problems
* Slow boot issues

---

### Boot Tab

Controls how Windows boots.

---

#### Important Options

| Option      | Meaning                           |
| ----------- | --------------------------------- |
| Safe boot   | Starts Windows in Safe Mode       |
| No GUI boot | Removes Windows loading animation |
| Boot log    | Saves startup logs                |
| Timeout     | How long boot menu waits          |

---

#### Safe Boot Types

| Type            | Purpose                       |
| --------------- | ----------------------------- |
| Minimal         | Basic Safe Mode               |
| Network         | Safe Mode with internet       |
| Alternate shell | Safe Mode with Command Prompt |

---

#### Example

If Windows is broken:
Open ``msconfig`` -> Go to ``Boot`` -> Check ``Safe boot`` -> ``Restart PC``

Now Windows starts in Safe Mode.

---

### Services Tab

This tab shows all Windows services.

A **service** is a background program.

Examples:
* Windows Update
* Antivirus
* Print Spooler
* DHCP Client

---

#### What You Can Do

| Action          | Purpose                   |
| --------------- | ------------------------- |
| Enable service  | Start service on boot     |
| Disable service | Stop service from loading |

---

### Important Checkbox

#### “Hide all Microsoft services”

Very important for troubleshooting.

Why?

* Prevents disabling critical Windows services accidentally.

---

### Example Troubleshooting

If a third-party app causes crashes:
- Hide Microsoft services
- Disable remaining services
- Restart
- Test system

This is called a **Clean Boot**.

# 4. Startup Tab

This tab normally manages startup applications.

BUT:

On newer Windows:

* It redirects to Task Manager.

Windows Server handles startup differently.

You usually check startup apps using:

```cmd

```

* Startup folder

## Task 3: Change UAC Settings

### What UAC Does
User Account Control protects Windows by asking for permission whenever apps or users try to make system‑level changes. The slider lets you choose how strict those notifications are.


### The Four Levels
| **UAC Notification Level**                                               | **Description**                                                                                                                                             |
| ------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Always notify**                                                        | Provides the highest level of security by notifying you whenever an application or you attempt to make changes to Windows settings.                         |
| **Notify me only when apps try to make changes (Default)**               | Recommended setting. Notifies you only when applications attempt to make changes to your computer, but not when you make changes yourself.                  |
| **Notify me only when apps try to make changes (do not dim my desktop)** | Similar to the default setting, but the desktop is not dimmed. Less secure because malware may be able to interact with the notification.                   |
| **Never notify**                                                         | Disables User Account Control (UAC). No prompts are displayed when changes are made, increasing the risk of unauthorized or malicious system modifications. |


### Purpose of User Account Control (UAC)
| **Feature**                    | **Purpose**                                                                                                                                                 |
| ------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **User Account Control (UAC)** | Protects Windows by requiring permission or administrator credentials before applications can make changes that affect the operating system or other users. |
| **Always notify**              | Best for maximum security, especially on shared or high-risk computers.                                                                                     |
| **Default (Recommended)**      | Provides a balance between security and usability for most users.                                                                                           |
| **Notify without dimming**     | Improves convenience but offers less protection because other programs can interact with the desktop during the prompt.                                     |
| **Never notify**               | Not recommended except for specific testing or isolated environments, as it significantly reduces system security.                                          |

## Task 4: Computer Management

### What is computer Management?
**Computer Management** is a built‑in Windows administrative tool that brings together many system utilities into one console. It’s like a control center for managing your computer’s performance, users, storage, and background services.

| **Section**                   | **Tool**                                  | **Purpose**                                                                                                                 |
| ----------------------------- | ----------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| **System Tools**              | **Task Scheduler**                        | Automates tasks such as running applications, scripts, or maintenance jobs at scheduled times.                              |
|                               | **Event Viewer**                          | Displays system, security, and application logs to troubleshoot errors, warnings, and events.                               |
|                               | **Shared Folders**                        | Manages shared folders, active user sessions, and files currently opened over the network.                                  |
|                               | **Local Users and Groups**                | Creates, modifies, and manages local user accounts and security groups.                                                     |
|                               | **Performance Monitor**                   | Monitors system performance, including CPU, memory, disk, and network usage.                                                |
|                               | **Device Manager**                        | Installs, updates, enables, disables, and troubleshoots hardware devices and drivers.                                       |
| **Storage**                   | **Disk Management**                       | Creates, deletes, extends, shrinks, and formats disk partitions, and assigns drive letters.                                 |
|                               | **Windows Server Backup** *(Server only)* | Creates and restores backups of the operating system, files, folders, and applications.                                     |
| **Services and Applications** | **Services**                              | Starts, stops, restarts, and configures background services, including their startup type (Automatic, Manual, or Disabled). |

#### Services List Columns
| **Column**       | **Description**                                                                                    |
| ---------------- | -------------------------------------------------------------------------------------------------- |

| **Log On As**    | Displays the account used to run the service (e.g., Local System, Local Service, Network Service). |

#### Common Service Startup Types
| **Startup Type**              | **Purpose**                                              |
| ----------------------------- | -------------------------------------------------------- |
| **Automatic**                 | Starts automatically when Windows boots.                 |
| **Automatic (Delayed Start)** | Starts automatically after Windows has finished booting. |
| **Manual**                    | Starts only when required by Windows or the user.        |
| **Disabled**                  | Prevents the service from starting until it is enabled.  |

#### Common Service Management Tasks
| **Task**                | **Purpose**                                                                                        |
| ----------------------- | -------------------------------------------------------------------------------------------------- |
| **Start Service**       | Starts a stopped service.                                                                          |
| **Stop Service**        | Stops a running service.                                                                           |

| **Pause Service**       | Temporarily suspends a running service (if supported).                                             |
| **Resume Service**      | Continues a paused service.                                                                        |
| **Change Startup Type** | Configures how the service starts (Automatic, Manual, or Disabled).                                |

#### Common Service Accounts
| **Account**         | **Purpose**                                                                                                    |
| ------------------- | -------------------------------------------------------------------------------------------------------------- |
| **Local System**    | Has extensive privileges on the local computer and is used by critical Windows services.                       |
| **Local Service**   | Runs with limited local privileges and accesses network resources anonymously.                                 |
| **Network Service** | Runs with limited local privileges but can authenticate to network resources using the computer's credentials. |

### Microsoft System Information (msinfo32)

System Information (msinfo32) is a built-in Windows tool that shows everything about your computer in one place.

Think of it as your computer's information book or health report. It helps you see what hardware and software your PC is using, making it useful for troubleshooting problems.

## Task 6: Resource Monitor

### Resource Monitor (resmon)

Resource Monitor (resmon) is a built-in Windows tool that lets you monitor how your computer is using its resources in real time.

Think of it as a live dashboard showing which programs are using your CPU, memory, disk, and network.

Why use Resource Monitor?

We can use it to:

- Find programs using too much CPU
- Check RAM (memory) usage
- See which files are being read or written
- Monitor internet and network activity
- Troubleshoot slow performance
- Identify processes causing high resource usage


#### The Four Main Sections
| **Section** | **What It Monitors**                         | **Example**                   |
| ----------- | -------------------------------------------- | ----------------------------- |
| **CPU**     | Processor usage by each program              | Chrome using 20% CPU          |
| **Memory**  | RAM usage by each program                    | Chrome using 2 GB RAM         |
| **Disk**    | Read/write activity on the hard drive or SSD | Copying a file to the SSD     |
| **Network** | Internet and network traffic                 | Downloading a file at 10 Mbps |




Press `Ctrl + S` -> after search:
````
resmon
````
