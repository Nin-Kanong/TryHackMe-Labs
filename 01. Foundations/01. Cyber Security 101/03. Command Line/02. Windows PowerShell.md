### What is PowerShell?
**PowerShell** is Microsoft's powerful command-line shell and scripting language used to manage Windows systems.

Unlike Command Prompt (CMD), PowerShell can:
- Manage Windows
- Automate tasks
- Control services
- Work with files
- Query system information
- Write automation scripts

### Learning Objectives

## What is PowerShell?



### What is PowerShell?

<img width="412" height="238" alt="image" src="https://github.com/user-attachments/assets/883f3265-9871-4872-82d6-d106c07428e6" />

**PowerShell* is a command-line shell and scripting language created by Microsoft to automate tasks and manage Windows systems.

It is much more powerful than Command Prompt (CMD).


### What Can PowerShell Do?
| Feature                  | Description                                    |
| ------------------------ | ---------------------------------------------- |
| Command-line Shell       | Run commands to manage Windows.                |
| Scripting Language       | Create scripts (`.ps1`) to automate tasks.     |
| Configuration Management | Configure and manage computers.                |
| Cross-Platform           | Runs on **Windows**, **Linux**, and **macOS**. |


### Brief History
| Year        | Event                                                                             |
| ----------- | --------------------------------------------------------------------------------- |
| Early 2000s | Microsoft created PowerShell to replace the limitations of CMD and batch scripts. |
| 2006        | First release of Windows PowerShell.                                              |
| 2016        | PowerShell Core released as open source for Windows, Linux, and macOS.            |

- ``Jeffrey Snover``, a Microsoft engineer, realised that Windows and Unix handled system operations differently—Windows used structured data and APIs, while Unix treated everything as text files. 

### Why PowerShell is Powerful
Unlike CMD, which outputs plain text, PowerShell works with objects.

#### CMD
Check IP:
````
ipconfig
````

#### PowerShell
````
Get-Process
````
Each process contains useful information like:
- Name
- ID (PID)
- CPU
- Memory
- Status


### What is an Object?
| Part           | Description                    | Example                  |
| -------------- | ------------------------------ | ------------------------ |
| **Properties** | Information about the object   | Name, Model, Color       |
| **Methods**    | Actions the object can perform | Start(), Stop(), Drive() |

Example: A car object
| Properties | Methods    |
| ---------- | ---------- |
| Color      | Drive()    |
| Model      | HonkHorn() |
| FuelLevel  | Refuel()   |

### 3. What is a Cmdlet?
A **cmdlet** is a small PowerShell command.
````
Verb-Noun
````
Command:
| Cmdlet         | Purpose                |
| -------------- | ---------------------- |
| `Get-Content`  | Read a file            |
| `Set-Location` | Change directory       |
| `Get-Date`     | Show current date/time |
| `Get-Process`  | Show running processes |
| `Get-Service`  | Show services          |


### 4. List All Commands
````
Get-Command
````
Shows:
- Cmdlets
- Functions
- Aliases
- Scripts


### 5. Filter Commands
Show only functions:
````
Get-Command -CommandType Function
````

Other examples:
````
Get-Command -CommandType Cmdlet
````


````
Get-Command -CommandType Alias
````


### 6. Get Help
Show manual for a command:
````
Get-Help Get-Date
````


Examples only:
````
Get-Help Get-Date -Examples
````

Detailed help:
````
Get-Help Get-Date -Detailed
````


Full documentation:
````
Get-Help Get-Date -Full
````


Online Microsoft documentation:
````
Get-Help Get-Date -Online
````
